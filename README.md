# e-plantShopping - Paradise Nursery

## Descripción del Proyecto
Este es el proyecto final del curso de React, denominado **e-plantShopping**. Es una aplicación de comercio electrónico diseñada para **Paradise Nursery**, una tienda especializada en plantas de interior que busca promover un estilo de vida más verde y saludable.

## Características principales
* **Landing Page:** Una página de bienvenida atractiva con una descripción de la misión de la empresa y un botón para comenzar la compra.
* **Listado de Productos:** Catálogo de plantas categorizado (Aire Puro, Aromáticas, etc.) con imágenes, descripciones y precios.
* **Carrito de Compras (Redux):** Implementación de un estado global utilizando Redux Toolkit para gestionar la adición, eliminación y actualización de cantidades de plantas.
* **Interfaz Responsiva:** Diseño adaptable para dispositivos móviles y escritorio.

## Tecnologías Utilizadas
* **React:** Librería principal para la interfaz de usuario.
* **Redux Toolkit:** Para la gestión del estado global del carrito.
* **Vite:** Herramienta de construcción y servidor de desarrollo.
* **CSS3:** Estilos personalizados para cumplir con el diseño del PDF de requerimientos.
